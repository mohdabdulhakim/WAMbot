# Vision

WAMbot is an AI-powered customer engagement, marketing automation, and conversational commerce platform designed primarily for Shopify and WooCommerce stores.

The platform enables businesses to automate WhatsApp marketing, customer support, customer retention, and AI conversations without requiring technical expertise.

The long-term goal is to become the most intelligent WhatsApp marketing platform for eCommerce.

---

# Core Philosophy

WAMbot is NOT another bulk WhatsApp sender.

It is an AI-powered customer engagement platform.

Every feature must increase one or more of the following:

• Customer acquisition
• Customer engagement
• Customer retention
• Customer lifetime value
• Revenue
• Automation

# Customer Data Strategy

WAMbot does NOT store complete order history.

Instead it stores:

Customer Information

Purchase Summary

Customer Statistics

Lifetime Value

Average Order Value

Order Count

Last Purchase

First Purchase

Tags

Imported Segments

This significantly reduces infrastructure costs while allowing advanced customer segmentation.
