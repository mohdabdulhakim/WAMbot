# MVP Definition

The MVP is complete when a business can:

1. Create an account and workspace.
2. Connect WhatsApp.
3. Receive and manually reply to messages.
4. Manage contacts.
5. Add its own OpenAI API key.
6. Create an AI Agent.
7. Give the AI Agent instructions and knowledge.
8. Test the AI Agent.
9. Activate the AI Agent for WhatsApp conversations.
10. Create basic automated workflows.
11. Send WhatsApp broadcast campaigns.
12. View basic performance analytics.

The MVP must prioritize reliability over feature quantity.
